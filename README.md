# 地球Online》游戏创意设计文档 v0.5

## 一、项目概况

### 1.1 游戏名称

**地球Online**（暂定）
### 1.2 一句话描述

一款基于AI驱动的、真实模拟当代中国年轻人生活的文字叙事型人生模拟游戏。
### 1.3 核心理念

让玩家在一个和现实运行机制完全相同的虚拟世界里，过一次**不一样但极其真实的人生**。
游戏不是逃避现实的工具，而是一面**映照内心的镜子**。通过AI生成的无尽可能，让玩家在“社会断奶期”（20-30岁）的日常琐碎、职场压力、家庭羁绊中，找到共鸣，引发思考。
### 1.4 风格基调

- **关键词**：沉默、内省、孤独、迷茫、真实、共鸣、科幻终端、复古未来主义
- **视觉语言**：极简、低饱和深色背景、文字优先、等宽字体、扫描线、粒子、终端绿/琥珀色
- **参考情绪**：
	>“真冬……这里是哪里？”
	“我呐，你说世界上真的有外星人吗？”

### 1.5 目标受众

正在经历或即将经历“社会断奶期”的年轻人（20-30岁），对人生感到迷茫、渴望被理解、喜欢深度叙事和哲学思考的玩家。

## 三、核心创意详细说明

### C1：AI驱动的真实NPC

**创意概述**：游戏中的关键人物（父母、上司、室友、相亲对象）全部由大语言模型驱动。他们不是念台词的工具人，而是有记忆、有情绪、会主动发消息的“数字生命”。
**核心体验**：
- 老妈发来“在吗？”，你知道接下来要谈考公或相亲
- 甲方说“感觉不对，但我也说不出哪里不对”，你必须应对
- NPC会记住你说过的话，一个月后还能提起

**技术支撑**：大语言模型API + 提示词工程 + 向量数据库记忆系统


----
### C2：S.A.V.E. 属性体系

**创意概述**：摒弃传统的力量/智力加点，用四个维度定义角色。初始天赋总值恒定，高智商必须用低家庭背景或差健康来换。


|属性|全称|描述|高值代价|低值优势|
|:-:|:-:|:-:|:-:|:-:|
|**S**|Social|社会关系、家庭背景|家庭期望高、束缚多|自由、无负担|
|**A**|Attribute|先天资质|容易被孤立、过劳|更懂人情冷暖|
|**V**|Vitality|生命活力|被期望一直坚强|更早学会自我照顾|
|**E**|Event|时运（隐藏）|机遇多，风险也大|平淡中的小确幸|


----
### C3：场景化主界面 + 虚拟手机

**创意概述**：游戏主界面不是聊天框，而是你当前所在地的场景（如出租屋的一角），随时间、天气变化光影。唯一的交互入口是一部“游戏内的手机”。
**核心体验**：
- 你看着窗外下雨，出租屋里只有手机屏幕亮着
- 手机里有聊天App、日记App、钱包App、招聘App
- 老妈的消息、花呗的账单、甲方的需求，都从这个手机涌进来

**设计意图**：最大化孤独感和沉浸感，同时结构清晰，玩家学习成本低。


----
### C4：叙事化初始化流程

**创意概述**：角色创建被包装成“未来生命重构终端”。你不是在填表，而是在恢复一份被碎片化的个人档案。6个章节、20个问题，每个选择都映射到深层游戏数据。
**章节结构**：
1. 时空锚点（出身年代、家庭背景）
2. 原生反馈（童年记忆、家庭关系）
3. 肉身容器（身体状况、精力水平）
4. 技能模组（天赋倾向、学习方式）
5. 灵魂残响（恐惧、渴望、人生哲学、外星人问题）
6. 初始锚点（名字、职业状态、开局道具）

**视觉风格**：终端绿/琥珀色文字、扫描线、全息档案卡、AI生成的总结文本


----
### C5：压力与代偿机制

**创意概述**：工作受挫、经济窘迫、家庭矛盾会累积“压力值”。高压力下解锁的“解压方式”往往是双刃剑。
**代偿行为示例**：


|行为|短期效果|长期代价|
|:-:|:-:|:-:|
|报复性熬夜刷短视频|快乐+10|精力上限永久-1|
|暴饮暴食高油高糖外卖|快乐+5|健康-2，体重+1|
|冲动网购9.9元垃圾袋|满足感+3|金钱-9.9，获得无用物品|
|给父母报喜不报忧|维护家庭关系|消耗自身情绪能量|
**积极代偿**（需要主动选择，消耗更多资源）：
- 运动、阅读、与朋友深度交流


----
### C6：真实社会规则模拟

**创意概述**：游戏世界运行着和现实一样的规则。你需要赚钱、花钱、面临各种不可逃避的社会压力。
**模拟系统清单**：


|系统|具体内容|
|:-:|:-:|
|**经济系统**|工资、房租（强制扣除）、花呗/信用卡还款、社保缴纳|
|**职场系统**|AI上司的PUA、深夜消息、无意义的加班、同辈竞争|
|**家庭系统**|催婚、催考公、转发养生谣言、家族群的压力|
|**信息网**|口口相传、社交媒体焦虑、信息的真假混杂|
|**健康系统**|熬夜的累积伤害、心理健康的量化、医保的使用|


----
### C7：记忆与滤镜系统

**创意概述**：你的日记不是客观记录，而是AI根据你当前情绪生成的“滤镜版”回忆。幸福时童年是金色，抑郁时同样的记忆变成灰色。
**核心体验**：
- 你需要翻看日记，才能确认“过去的真相”
- 同一件事，在不同时间回忆，文字完全不同
- 这是治愈自我的过程，也是区分“事实”与“感受”的练习


----
### C8：隐藏哲学问题

**创意概述**：源自你提供的第二张图。“你相信世界上有外星人吗？”这个问题不直接映射数值，但会影响AI在哲学性场景中的倾向。
**设计意图**：
- 在某个加班到深夜的夜晚，系统主动问出这句话
- 这是一个**存档点**，也是游戏对玩家的关怀：“你现在，还好吗？”
- 成为玩家社群中口口相传的彩蛋

## 四、UI/UX创意设计

### 4.1 整体美术风格

- **主题**：复古未来主义的全息操作系统
- **主色调**：深黑底 + 终端绿（`#33ff33`）或琥珀色（`#ffb347`）
- **字体**：等宽字体（JetBrains Mono / Courier New），中文细等线
- **装饰元素**：扫描线、数据粒子、CRT屏幕闪烁、状态指示灯、波形图
- **动画风格**：文字打字机/刷入效果、缓慢平滑的过渡、微弱的呼吸灯

### 4.2 界面层级架构

text
```
L0: 加载页面（氛围营造 + 资源预加载）
L1: 主菜单（全息终端入口）
L2: 初始化流程（档案重构，6章节20问）
L3: 游戏主界面（场景背景 + 虚拟手机覆盖层）
    ├── 聊天App
    ├── 日记App
    ├── 钱包App
    ├── 招聘App
    └── 设置App
```
### 4.3 页面清单

## 五、技术实现创意点

### 5.1 AI记忆系统

- **短期记忆**：当前对话上下文窗口
- **长期记忆**：向量数据库存储关键事件，相似度检索
- **摘要记忆**：每次对话后AI自动生成一句话摘要
- **渐进策略**：初期用纯文本摘要，数据量大后再引入向量检索

### 5.2 游戏内“手机”组件

- 不是系统手机，而是HTML/CSS绘制的独立组件
- 内部多“App”通过状态切换实现路由
- 聊天界面支持打字机效果、“对方正在输入…”提示

### 5.3 世界状态机（后端）

- 定时任务驱动：自动扣房租、触发随机事件、更新经济数据
- AI生成管道：输入（玩家数据+记忆+世界状态）→ AI处理 → 输出（文本+游戏指令）

### 5.4 技术栈创意

- **前端**：HTML/CSS/JS + 响应式设计，优先手机竖屏
- **后端**：Python + FastAPI（学习机会，AI生态最优）
- **数据库**：PostgreSQL + pgvector（关系型数据 + 向量记忆，零额外服务）
- **AI**：大语言模型API + 提示词工程
